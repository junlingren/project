Healthcare Fraud Detector
To reduce the fraud claim risk, developed a Fraud Detector to detect the potential fraudulent provider. Cleaned
and processed imbalanced medicare claims datasets. Applied Matplotlib and Seaborn to visualize data.
Optimized multiple classification models (Logistic Regression, Random Forest, and XGBoost). XGBoost
classifier (ROC-AUC: 0.86 and recall: 0.83) successfully identified key fraud indicators and revealed the
characteristics of fraudulent providers to generate fraud preventative strategies.
Wrapped the optimized model as an API via Flask and deployed it on the Google Cloud Platform (GCP).
